Visibilidad El front-end es la parte visible de un sitio web o aplicación, mientras que el back-end es la parte que no se puede ver. Interacción El front-end es lo que los usuarios ven e interactúan, mientras que el back-end es la infraestructura que permite que la aplicación funcione. Lenguajes Los lenguajes utilizados para el front-end son HTML, CSS, JavaScript, mientras que los utilizados para el back-end incluyen Java, Ruby, Python, .Net. AUTOR: Carlos
